# 🤖 Sistema Inteligente de IA

> Sistema completo de IA con búsqueda web, RAG y evaluación automática de modelos

## 🚀 Inicio Rápido

### Backend
```bash
pip install -r configuraciones/requirements.txt
python -m servidor.main
```

### Frontend
```bash
npm install
npm run dev
```
